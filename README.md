# java-app-imagej-framework-demo
**A simple Java Image Processor app with ImageJ Framework.**

## Current features
- Image Type Conversion
- Brightness and Contrast adjuster
- Threshold adjuster
- Process Filters and Shadows
- Particle Analysis

## Upcoming features
- Histogram Analysis and Editing
- Image Stiching
